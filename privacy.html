Privacy Policy

Last updated: March 14, 2026

1. Data Collection

TikTok AI Generator is a personal automation tool. We collect only the minimum data required to authenticate with TikTok and publish content on your behalf:

  • TikTok OAuth access token (stored locally on your device)
  • Basic profile information (display name, open ID) for account verification

2. Data Storage

All data is stored locally on your device. No data is transmitted to external servers other than TikTok's API for authentication and content publishing.

3. Third-Party Services

This tool interacts with the following services:

  • TikTok API - for authentication and video publishing
  • Microsoft Edge TTS - for text-to-speech generation
  • Replicate - for AI image generation
  • Anthropic - for AI script generation

Each service has its own privacy policy that applies to data processed through their platforms.

4. Data Sharing

We do not sell, share, or distribute any user data to third parties. We do not collect data from other TikTok users.

5. Data Deletion

Since all data is stored locally, you can delete it at any time by removing the application files and revoking TikTok access in your account settings.

6. Contact

For questions about this privacy policy, contact the application maintainer.
